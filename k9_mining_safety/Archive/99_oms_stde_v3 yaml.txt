# de observaciones, incidentes y auditorías a lo largo del STDE v3.
# Se generan como series continuas normalizadas entre 0 y 1.
#
# La narrativa oficial establece que:
# - FDO climáticos, de fatiga y congestión aumentan progresivamente.
# - Backlog, dotación y variabilidad influyen en señales OCC y NMS.
# - La combinación de FDO produce degradación silenciosa (R01) y
#   degradación acumulativa dominante (R02), coherente con Texto 3 y 4.
# ================================================================

    se genera de forma independiente y normalizada en el rango 0–1. En la
    narrativa STDE v3, los FDO son el motor principal de la variabilidad y

# ================================================================
# Observaciones (OPG y OCC) y estados valorativos (+/-)
# ================================================================
# Las observaciones son señales de conducta y condición registradas en
# terreno. STDE v3 utiliza:
#   - OPG: Observaciones Preventivas Generales.
#   - OCC: Observaciones de Control Crítico.
#
# Cada tipo puede expresarse en dos estados valorativos:
#   - OPG+ / OPG-  → gestión adecuada / gestión débil.
#   - OCC+ / OCC-  → control crítico cumplido / fallido.
#
# Estas señales representan la “alerta temprana” del sistema y son
# esenciales para explicar degradación silenciosa (R01) y degradación
# acumulada dominante (R02), tal como describe la narrativa oficial.
# ================================================================
